# CLMandarin

Offline-first flashcard web app for studying HSK 1-6 Mandarin Chinese vocabulary.

## Features

- **Browse HSK 1-6 vocabulary** — 5,000 words with hanzi, pinyin, and English translations
- **Create flashcard lists** — Like Spotify playlists, but for Chinese words
- **Add words or entire HSK levels** — Star individual words or bulk-add by level
- **Reorder cards** — Drag and drop to arrange your study order
- **Study mode** — Flip through cards one by one
- **Save & load lists** — Export as JSON files, import from files or folders
- **Works offline** — PWA with full offline support, no internet needed after first load

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Styling**: TailwindCSS
- **Storage**: IndexedDB (via Dexie.js)
- **Offline**: PWA with Service Worker (vite-plugin-pwa)
- **Data**: Vocabulary scraped from [MandarinBean](https://mandarinbean.com)

## Getting Started

```bash
# Install dependencies
npm install

# Run the scraper (generates vocab JSON files)
npm run scrape

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Data

Vocabulary data is scraped from MandarinBean's HSK word lists and stored as static JSON in `public/data/`. Word counts per level:

| Level | Words |
|-------|-------|
| HSK 1 | 150 |
| HSK 2 | 150 |
| HSK 3 | 300 |
| HSK 4 | 600 |
| HSK 5 | 1,300 |
| HSK 6 | 2,500 |

## File Format

Flashcard lists are saved as `.clmandarin.json` files:

```json
{
  "version": 1,
  "exportedAt": "2026-02-27T00:00:00.000Z",
  "list": {
    "id": "uuid",
    "name": "My HSK 1 Basics",
    "wordIds": ["hsk1-001", "hsk1-002"]
  },
  "words": [
    { "id": "hsk1-001", "hskLevel": 1, "hanzi": "我", "pinyin": "wǒ", "english": "I, me" }
  ]
}
```
